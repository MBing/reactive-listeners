# JS Boilerplate 2019

## Dependency (You need to have this on your system installed.)
- [yarn](https://yarnpkg.com/lang/en/)
- [node](https://nodejs.org/en/)

## Usage

    yarn install
    
    yarn build # production build
    
    yarn dev   # development build with server & hot reloading
